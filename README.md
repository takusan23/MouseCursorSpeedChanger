# MouseCursorSpeedChanger
Windows 10 のマウスカーソルの速度を切り替えるWPFアプリ。  
（マウス速度は5/タッチパッド速度は10）

# ダウンロード
https://github.com/takusan23/MouseCursorSpeedChanger/releases/tag/1.0

# つかいかた
exeを起動すれば**タスクトレイ**にアイコンが表示されます。  
アイコンを右クリックすることでコンテキストメニューが出るので切り替えられます。  
終了もコンテキストメニューにあります。
